# CUDA Image Resize

A CUDA C++ project for fast image resizing on the GPU.

This example is useful for learning coordinate mapping, pixel sampling, and image scaling in CUDA.

## Features

- image resizing with CUDA
- GPU-based scaling operation
- beginner-friendly geometric image processing example
- clean C++ project structure

## Tech Stack

- C++
- CUDA
- OpenCV

## Project Goal

This repository demonstrates:

- how image coordinates are remapped on GPU
- how parallel threads process scaled images
- how CUDA can accelerate geometric transformations

## Future Improvements

- bilinear interpolation
- batch image resize
- video stream resize
- Jetson optimization
